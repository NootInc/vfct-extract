# vfct-extract

Rust program that extracts the VBIOS from an AMD VFCT ACPI table

The syntax for this small program is `executable <INPUT FILE> <OUTPUT FILE>`, i.e. `cargo run ~/Downloads/vfct.dat ~/Downloads/VBIOS.bin`
